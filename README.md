# PostAI

🚀 An AI-powered backend application that generates captions from uploaded images using Gemini AI.

## Features

- 📸 Image Upload
- 🔐 JWT Authentication
- 🤖 AI Caption Generation
- ☁️ Cloud Storage Integration
- 💾 MongoDB Database

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Multer
- Gemini AI

## Workflow

Upload Image → Authenticate User → Generate AI Caption → Save Data to MongoDB
